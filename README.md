## 프로젝트 목적
  - 중고거래가 활성화 되어있어, 중고가격을 추천해주는 웹 어플리케이션을 개발하고자 함
 
## 프로젝트 개요
- 중고거래 사이트(당근마켓, 번개장터)에서 특정 상품에 대한 정보 스크래핑
- 스크래핑으로 수집한 데이터를 전처리
- Catboost 모델을 이용한 상품 관심도 예측
- 관심도에 따른 예상가격 추천

## 예측모델 측정지표
- R^2 : -0.1 -> 0.3

## 데이터
- 당근마켓 스크래핑
- 번개장터 스크래핑
